Lane Detection and Depth Estimation

Link

The overall goal is to make something akin to “Artificial General Intelligence”. The idea is that you can give an “endpoint” to the program and it will be able to navigate to it on its own. The project is written in Python and uses Tensorflow and PyTorch. The next step is to transition the project to PyTorch and eventually to have a planning network that uses the depth estimation to construct a vector space model through which it can plan a path.

Ben's Revenge Scratch Game

Link

This was created when I took Intro to Programming and Robotics and it’s a scratch game where the objective is to hit bears with your sword. If the bear touches you, then you lose health. It incorporates some APCS-like concepts, like continuously checking the amount of health you have and what stage you’re on by using a forever loop and broadcasting events to other sprites
